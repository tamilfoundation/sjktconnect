# Sprint 3.5 — Tamil Translation Review + Deployment

## Objective
Review and fix Tamil translations against the style guide, then deploy backend and frontend to production with all Sprint 3.3-3.4 changes.

## Part 1: Tamil Translation Review
Audit ~162 Tamil strings in `messages/ta.json` against the Tamil style guide (85+ grammar rules).

**Checks:**
- Vallinam doubling after அந்த/இந்த/எந்த, infinitives, adverbs, case markers
- JOIN vs SEPARATE for postpositions, auxiliary verbs, போது/வரை/வேண்டும்
- Natural phrasing (not machine-transliterated)
- Consistent terminology

**Output:** Fixed `ta.json`, tests passing.

## Part 2: Deploy Backend
1. Verify gcloud config (account: admin@tamilfoundation.org, project: sjktconnect)
2. `gcloud run deploy sjktconnect-api --source . --region asia-southeast1 --allow-unauthenticated`
3. Verify: `/health/` + `/api/v1/schools/national-stats/`

## Part 3: Deploy Frontend
1. `gcloud run deploy sjktconnect-web --source . --region asia-southeast1 --allow-unauthenticated`
2. Verify: homepage hero, about page, language switcher, school profile labels, footer

Standard sprint close workflow.
